Pemrograman Javascript: Penulisan Kode Pada javascript

Penulisan Kode Javascript

JavaScript merupakan bahasa scripting pada sisi client yang dijalankan dalam web browser. Untuk memanggilnya digunakan tag <script> dan ditutup dengan tag </script>

Tag script ini dapat ditempatkan secara internal dalam tag <body> atau tag <head> dalam halaman HTML. Tidak hanya itu, script juga bisa diletakkan secara terpisah dalam file eksternal

1. Internal JavaScript dalam <head>

HTML
<!DOCTYPE html>
<html>
<head>
	<title>Belajar Javascript</title>
	<script>
		function myFunction(){
			document.getElementById("paragraf").innerHTML = "Hypertext Markup Language";
		}
	</script>
</head>
<body>
	<p id="paragraf">HTML</p>
	<button type="button" onclick="myFunction()">Show Abbr</button>
</body>
</html>

Jika dijalankan pada browser hasilnya sebagai berikut:

Ketika tombol Show Abbr di klik akan mengganti teks HTML dengan Hypertext Markup Language karena pengaruh procedure myFunction()

2. Internal JavaScript dalam <body>

HTML
<!DOCTYPE html>
<html>
<head>
	<title>Belajar Javascript</title>
</head>
<body>
	<script>
		document.write = "Hello World";
	</script>
</body>
</html>

3. Eksternal JavaScript

HTML
<!DOCTYPE html>
<html>
<head>
	<title>Belajar Javascript</title>
	<script src="js/myScript.js">
	</script>
</head>
<body>
	<p id="paragraf">HTML</p>
	<button type="button" onclick="myFunction()">Show Abbr</button>
</body>
</html>
JS
function myFunction(){
	document.getElementById("paragraf").innerHTML = "Hypertext Markup Language";
}

Ketika dijalankan di web browser hasilnya seperti internal javascript dalam <body>

Keuntungan penggunaan Eksternal Javascript adalah sebagai berikut:

  • Memisahkan HTML dan code
  • Membuat HTML dan Javascript lebih mudah dibaca dan dimaintain
  • File Cache Javascript dapat mempercepat loading halaman

4. Atribut Event

Cara yang ketiga ini sering digunakan untuk memanggil fungsi pada event terentu. Misalnya ketika suatu elemen diklik, maka jalankan fungsi Javascript.

JS
<button onclick="alert('Ok Terima kasih!')">Klik donk!</button>

Selain event onclick ada juga event-event yang lain, seperti onsubmit, onload, ondoubleclick, onmouseover, onmouseout, dsb.
